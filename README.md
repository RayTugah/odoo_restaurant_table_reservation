# Restaurant Table Reservation

Odoo 19 addon for restaurant table reservations.

## Features

- Table reservation management.
- Tags for visual service indicators.
- Wizard for creating reservations.
- Date-based ticket printing wizard.
- Integrated 80 mm TicketMesas QWeb report.
- Helpers for migration from the previous Odoo Studio implementation.
- Reference integration for the Camping Fuente production workflow.

## Technical module name

```text
odoo_restaurant_table_reservation
```
